- LOOP:
  ----

> when we want to iterate over list of arguments or perform same operation with large amount of data 
  
  FOR LOOP:
  --------

  for i in list;do (command) ; done

     OR

  for i in list
    do 
      echo $i
  done 


  eg:
  --
   for i in a b c d e;do echo $i;done

  OUTPUT: a b c d e
  ------
